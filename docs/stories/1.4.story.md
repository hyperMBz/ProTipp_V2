## 🔄 Auto-refresh Rendszer

### **Konfiguráció**
- **Default interval**: 30 másodperc
- **Configurable** per widget
- **Visual feedback** loading állapotokkal
- **Error handling** hálózati problémákhoz

### **Implementáció**
```tsx
useEffect(() => {
  if (!autoRefresh) return;
  
  const interval = setInterval(() => {
    handleRefresh();
  }, refreshInterval);
  
  return () => clearInterval(interval);
}, [autoRefresh, refreshInterval]);
```
